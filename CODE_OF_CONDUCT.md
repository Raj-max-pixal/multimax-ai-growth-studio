# 🌟 Our Core Values

| 💙 Value         | Description                                                      |
| :--------------- | :--------------------------------------------------------------- |
| 🤝 Respect       | Treat everyone with kindness, dignity, and professionalism.      |
| 💡 Innovation    | Encourage creative ideas and continuous improvement.             |
| 🌍 Inclusion     | Welcome contributors from every background and experience level. |
| 📚 Learning      | Share knowledge and help others grow.                            |
| 🚀 Collaboration | Work together with transparency and trust.                       |
| ❤️ Integrity     | Be honest, ethical, and responsible in all interactions.         |

---

# ✅ Expected Behaviour

We encourage every contributor to:

* 🤝 Treat everyone respectfully.
* 💬 Use welcoming and professional language.
* 💡 Provide constructive feedback.
* 📖 Share knowledge openly.
* ❤️ Support new contributors.
* 🚀 Collaborate positively.
* 🔍 Report issues responsibly.
* 🌍 Respect different perspectives.
* 🎯 Focus on improving the project.

# 🛠️ Contribution Standards

Before opening a Pull Request, please ensure that you:

* ✅ Follow the project's coding standards.
* ✅ Write clean, readable, and maintainable code.
* ✅ Test your changes thoroughly.
* ✅ Update documentation when necessary.
* ✅ Keep commits meaningful and descriptive.
* ✅ Respect feedback during code reviews.

---

<details>
<summary><strong>📢 Reporting Issues</strong></summary>

If you experience or witness unacceptable behaviour:

* Open a confidential GitHub Issue (if appropriate).
* Contact the project maintainer.
* Include relevant details or evidence whenever possible.

Every report will be reviewed fairly, respectfully, and confidentially.

</details>

---

<details>
<summary><strong>⚖️ Enforcement</strong></summary>

Project maintainers are responsible for enforcing this Code of Conduct.

They may:

* Edit or remove inappropriate content.
* Reject pull requests that violate these guidelines.
* Suspend repeat offenders.
* Permanently ban severe violations.

Our priority is maintaining a safe, welcoming, and productive community.

</details>
